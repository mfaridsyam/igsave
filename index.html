Cara ambil Session ID:
1. Buka Instagram di browser desktop (chrome/firefox)
2. Login ke akun Instagram kamu
3. Tekan F12 → tab Application
4. Klik Cookies → https://www.instagram.com
5. Cari baris sessionid → copy nilainya

🔒 Session ID hanya disimpan di browser kamu, tidak dikirim ke server kami.
